The repo is structured around a few folders. 

The src folder contains all of the source code of the main part of the
project, written in C 

The city_graph dir contains the python script to retrieve the graphs from OSM /
pretreat them them and write them in a CSV format. 

the perftest directory contains the python script used to generate 
graphs showing the performances of the programs and the results of these tests

the test_graph dir contains a few graphs that I use for testing 

the csv_graph.tar.gz archive contains csv rep of graphs generated with the python script in 
city_graphs
